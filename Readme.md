# The Developer Interface // Archivo de desarrollador

¡Bienvenido al registro oficial del Sistema! Este es un portafolio web personal e interactivo inspirado en la interfaz de **Solo Leveling**, Se diseño para desplegar estadísticas de clase, habilidades de desarrollo y un almacén de recuerdos. (Nomas por mame xD)

## Tecnologías Utilizadas
* **HTML5** & **JavaScript (Vanilla)** para la lógica de interfaz.
* **Tailwind CSS** para los estilos futuristas, efectos de neón y animaciones globales.
* **Fuentes Digitales:** Orbitron y Share Tech Mono.

## Características del Sistema
* **Protocolo de Autenticación Falso:** Un sistema interactivo que evalúa al visitante (con un huevo de pascua incluido (Solo por joda xD)).
* **Estadísticas de Clase:** Atributos dinámicos medidos en niveles de energía/XP.
* **Almacén de Recuerdos:** Inventario de ítems con sistema de rarezas (Común, Raro, Épico, Legendario).
* **Escaneo de Pantalla Global:** Efecto láser de barrido optimizado para una experiencia inmersiva en laptops y móviles.
